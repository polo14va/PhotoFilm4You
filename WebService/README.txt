# Installation and Execution Instructions

1. Execute the file located at the root `start.sh` using the `-build` command.
2. If the Docker environment is not already running, execute the `start.sh` file again with the `-docker` command. This will perform `docker compose up` and start 3 Java Spring microservices that can be accessed from:

   - Product Catalog: [http://localhost:18081/swagger-ui/index.html](http://localhost:18081/swagger-ui/index.html)
   - User: [http://localhost:18082/swagger-ui/index.html](http://localhost:18082/swagger-ui/index.html)

3. Enjoy!


# Instrucciones de Instalación y Ejecución

1. Ejecutar el archivo ubicado en la raíz `start.sh` con el comando `-build`.
2. En caso de no tener el entorno de Docker levantado, ejecutar nuevamente el archivo `start.sh` con el comando `-docker`. Esto realizará `docker compose up` y levantará 3 microservicios Java Spring a los cuales se podrá acceder desde:

   - Product Catalog: [http://localhost:18081/swagger-ui/index.html](http://localhost:18081/swagger-ui/index.html)
   - User: [http://localhost:18082/swagger-ui/index.html](http://localhost:18082/swagger-ui/index.html)

3. ¡Disfruta!
